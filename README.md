# mccluney-family-tree

A beautiful, interactive website project to preserve and explore the McCluney family heritage, with active tools to research and discover more ancestors from public sources.

> "A project put in place by Byron McCluney to help other McCluney's find their heritage."

## Features

- Interactive Family Tree + Searchable Directory
- Research Hub with live WikiTree search and local Python crawler
- Admin-style dashboard for running research tasks from the UI
- Easy to extend with new verified data

## Getting Started

Open index.html or deploy via GitHub Pages / Vercel.

## Research Tools

Use the Research Hub in the site for live searches and the scripts/ folder for the full crawler.
